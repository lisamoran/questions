Questions App User Guide

The Questions App

The Questions App enables the user to select locations and set questions associated with these locations for use in the Quiz App.

The user can set a question and give four possible answers, as well as indicate which is the correct answer.

How do I use the Questions App?

1. First, the user should navigate around the map and click on the map at the location that they wish to put a question. A popup will appear which contains the latitude and longitude of the location.

2. The user should then open the menu on the left, by clicking on the menu button which is symbolised by three horizontal lines. This will reveal the question setting form.

3. Fill in the question creation form, by copying the latitude and longitude values displayed on the map popup, and typing in a question, four possible answers and the correct answer.

4. When the question creation form has been filled in, click on the 'Start Data Upload' button, to begin the upload of the question data.

5. When the upload process has been successfully completed, a message will apppear under the button saying "Your question has been added!"

Steps 1 to 5 can be repeated for a new question.
